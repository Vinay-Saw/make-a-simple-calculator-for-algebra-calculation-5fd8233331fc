# Simple Algebra Calculator

## Overview
This is a simple web-based algebra calculator that can take 2 or 3 variables and perform basic operations on them.

## Features
- Responsive design that works on both desktop and mobile.
- Clean and user-friendly interface.
- Supports input of 2 or 3 variables for calculations.

## Technology Stack
- HTML5
- CSS3
- JavaScript

## Setup Instructions
1. Clone the repository:
    ```bash
    git clone https://github.com/yourusername/algebra-calculator.git
    cd algebra-calculator
    ```
2. Open the `index.html` file in your web browser. No server is needed to run this application.

## Usage Instructions
1. Enter values in the input fields. The first two fields are required, while the third is optional.
2. Press the "Calculate" button to see the result.
3. The result will display below the button.
